Predator SDR

Lowering the barrier to entry for the electromagnetic environment

Built on SDR++

About

Predator SDR is a fork of SDR++, reimagined with a cleaner, more mission-focused interface. It is designed for operators who want to monitor and understand the EM environment without a steep learning curve.

The project currently targets Android as its primary platform, with Linux, Windows, and macOS builds on the roadmap.

Linux Build Instructions

# Install dependencies
sudo apt install cmake libfftw3-dev libglfw3-dev \
  librtlsdr-dev libhackrf-dev libairspy-dev \
  portaudio19-dev libsoapysdr-dev

# Build
mkdir build && cd build
cmake .. -DCMAKE_BUILD_TYPE=Release
make -j$(nproc)
sudo make install
